Admin Functions

These buttons below allow the contract deployer to stop the operation of the contract if something unexpected occurs. This might include exploiters utilizing a mistake made in the code, in which case the contract owner should fix. There are many instances in which this has occured creating widespan panic, such as the DAO attack.
